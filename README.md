# Orbital Reckoning

Vertical slice game 3D tren web, xay bang `Next.js` va `Three.js`.

## Co gi trong ban nay

- Dau truong 3D co camera chase va scene sci-fi day du.
- Player move, aim, fire, dash.
- Enemy AI gom scout, spitter, crusher.
- Pickup shield, score, wave progression, game over va restart.
- HUD responsive, co touch controls de mo tren mobile van dung layout.

## Chay local

```bash
npm install
npm run dev
```

Mo `http://localhost:3000`.

## Build production

```bash
npm run build
npm run start
```

## Dua len GitHub

```bash
git init
git add .
git commit -m "Create Orbital Reckoning 3D web game"
```

Sau do tao repo tren GitHub va push len.

## Deploy Vercel

```bash
vercel
```

Vercel se tu nhan dien day la project Next.js.
